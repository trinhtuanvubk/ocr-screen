### PaddleOCR
- To install:
```bash
python3 -m pip install paddlepaddle -i https://mirror.baidu.com/pypi/simple
pip install "paddleocr>=2.0.1
```

- Run a test:
```bash
python3 paddleocr_test.py
```

### MMOCR
- To install:
```bash
git clone https://github.com/open-mmlab/mmocr.git
cd mmocr
pip install -e .
```
